# 🚌 VegaGo

> **La app definitiva para moverte por la Vega Baja.**

Bienvenido al repositorio oficial de **VegaGo**. Este espacio sirve como referencia pública para el proyecto VegaGo, la plataforma de información de transporte líder en la comarca de la Vega Baja del Segura (Alicante).

---

## 🌐 Visítanos
Puedes acceder a la plataforma completa en:  
👉 **[https://vegago.app](https://vegago.app)**

---

## ✨ ¿Qué es VegaGo?
VegaGo es una solución digital diseñada para conectar a los ciudadanos de la Vega Baja con el sistema de transporte público. Nuestra misión es facilitar la movilidad regional mediante datos precisos y una interfaz moderna.
